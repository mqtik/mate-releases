Keyboard Shortcuts

Common keyboard shortcuts for Mate on desktop.

These shortcuts work on macOS. On Windows and Linux, replace Cmd with Ctrl.

General

ActionShortcut
New terminal tabCmd + T
Close current tabCmd + W
Switch to next tabCmd + Shift + ] or Ctrl + Tab
Switch to previous tabCmd + Shift + [ or Ctrl + Shift + Tab
Open settingsCmd + ,
Toggle sidebarCmd + B

Terminal

ActionShortcut
Copy selectionCmd + C
PasteCmd + V
Clear terminalCmd + K
Search in terminalCmd + F
Zoom in (increase font size)Cmd + =
Zoom out (decrease font size)Cmd + -
Reset zoomCmd + 0

Editor / IDE

ActionShortcut
Save fileCmd + S
FindCmd + F
Find and replaceCmd + H
UndoCmd + Z
RedoCmd + Shift + Z
Select allCmd + A

Agent Chat

ActionShortcut
Send messageEnter
New line in messageShift + Enter
Focus message inputCmd + L

Keyboard shortcuts are primarily available on desktop platforms (macOS, Windows, Linux). Mobile devices use touch gestures instead.

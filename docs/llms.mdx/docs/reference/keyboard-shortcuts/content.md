# Keyboard Shortcuts (/docs/reference/keyboard-shortcuts)



These shortcuts work on macOS. On Windows and Linux, replace **Cmd** with **Ctrl**.

## General [#general]

| Action                 | Shortcut                                  |
| ---------------------- | ----------------------------------------- |
| New terminal tab       | `Cmd + T`                                 |
| Close current tab      | `Cmd + W`                                 |
| Switch to next tab     | `Cmd + Shift + ]` or `Ctrl + Tab`         |
| Switch to previous tab | `Cmd + Shift + [` or `Ctrl + Shift + Tab` |
| Open settings          | `Cmd + ,`                                 |
| Toggle sidebar         | `Cmd + B`                                 |

## Terminal [#terminal]

| Action                        | Shortcut  |
| ----------------------------- | --------- |
| Copy selection                | `Cmd + C` |
| Paste                         | `Cmd + V` |
| Clear terminal                | `Cmd + K` |
| Search in terminal            | `Cmd + F` |
| Zoom in (increase font size)  | `Cmd + =` |
| Zoom out (decrease font size) | `Cmd + -` |
| Reset zoom                    | `Cmd + 0` |

## Editor / IDE [#editor--ide]

| Action           | Shortcut          |
| ---------------- | ----------------- |
| Save file        | `Cmd + S`         |
| Find             | `Cmd + F`         |
| Find and replace | `Cmd + H`         |
| Undo             | `Cmd + Z`         |
| Redo             | `Cmd + Shift + Z` |
| Select all       | `Cmd + A`         |

## Agent Chat [#agent-chat]

| Action              | Shortcut        |
| ------------------- | --------------- |
| Send message        | `Enter`         |
| New line in message | `Shift + Enter` |
| Focus message input | `Cmd + L`       |

<Callout type="info">
  Keyboard shortcuts are primarily available on desktop platforms (macOS, Windows, Linux). Mobile devices use touch gestures instead.
</Callout>
